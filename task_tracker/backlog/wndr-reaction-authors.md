# Авторы реакций WNDR

Сейчас собираем только суммарные реакции на пост (`{emoji, count}`), без авторов.
Telethon может получить авторов через `GetMessageReactionsListRequest`, но это отдельный запрос на каждое сообщение — для 7000+ постов упрёмся в rate limit.

**Подход:** собирать авторов только для постов с реакциями (или только для содержательных). Это даст граф "кто кому реагирует" — полезно для анализа взаимности и влияния.
